# yarder (React version)

React app designed by Alexandre Fleurquin.

## Project setup
```
npm install
```
```
bundle install
```

### Create database
```
rails db:create
```
```
rails db:migrate
```

### Compiles and hot-reloads for development
```
npm run dev
```
```
rails s
```

### Compiles and minifies for production
```
npm run build
```
